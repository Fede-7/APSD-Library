Getting Started
This project requires the following dependencies:
 * Programming Language: Java
 * Package Manager: Maven
Installation
Build APSD-Library from the source and install dependencies:
1. Clone the repository
git clone https://github.com/rede-J/APSD-Library

2. Navigate to the project directory
cd APSD-Library

3. Install the dependencies
Using Maven:
mvn install

Usage
Run the project with Maven:
mvn exec:java

Testing
APSD-Library uses the test_framework test framework. Run the test suite with Maven:
mvn test
